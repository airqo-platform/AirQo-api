{# Minimal fallback template for django_filters' browsable filter form #} Filter form

Filter UI (minimal)

This is a lightweight fallback for the django-filters form used by the DRF browsable API. The project did not ship the original template at django_filters/rest_framework/form.html, so a minimal helpful page is shown here instead.

To use filtering, add the appropriate query parameters to the request URL (for example: ?search=term or other filter names exposed by the endpoint). See the API docs for the exact fields and examples:

  • Interactive API (Redoc) — OpenAPI-driven documentation.
  • OpenAPI schema (raw) — machine-readable schema.
  • API docs — if available in your environment.

If you prefer the original django-filters UI, install a release of django-filter that ships the template, or copy your desired template into templates/django_filters/rest_framework/form.html in this project to customize the appearance.
